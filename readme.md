# Jared Chmielecki

## Vagrant
run from project root
```bash 
cd vagrant
vagrant up
open http://localhost:9090
```

## Docker Compose
run from project root. No port specified assumed to mean 80
```bash
cd docker
docker-compose up
open http://localhost
```
